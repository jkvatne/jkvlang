# JKV, a toy compiler for a new language

This is a compiler written completely from scratch.
The language is also completely new, even if it is 
heavily inspired by Go, which is currently my favorite
language. The compiler is also written in Go.

One of the goals are to learn why it is so difficult to 
avoid memory leaks and to research if the nice features
of Rust can be emulated in a simpler way.

Currently a lot of features are missing, but it is now
compiling and running all the test examples.

# Implemented functionality 

* Strings
* Dynamic arrays
* Structs
* Integer and float types
* Simple arithmetics for integers and floats
* File read/write

# Missing functionality

* Modules/packages
* Linking precompiled object files
* Most system calls
* All of the math libraries.
* Maps

# Warning

This is a single person project. I do not plan for others to contribute.
Nor will I fix problems for other persons. But the code is free for all.
